## âœ¨ Key Features
1. **User Registration & GDPR:**
   - Secure user sign-up with mandatory GDPR consent.
   - **Right to Access:** Users can view their stored personal data in the Profile section.
   - **Right to Erasure:** Documentation provided on data deletion policies.

2. **Event Application:**
   - Users can apply for specific events (e.g., Men's Singles, Caber Toss).
   - Smart routing redirects users to the application page upon login.

3. **Admin Dashboard:**
   - Role-based login (Admin vs User).
   - Admins can view a full list of registered users and their application status.

4. **Real-time Info:**
   - Interactive scoreboard and schedule displaying mixed event types.
